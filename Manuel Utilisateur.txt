Pour utilisation du projet:

	/Projet/Modele/
		infoCo.php => Information de connexion au serveur
		BDD.sql => La base de données
		BDDInsert.sql => Les insertion dans la base de données

	/Projet/Vue/
		style.css => Feuille de style gérant tout l'affichage
		* => Les différentes vue

	/Projet/Controleur/
		* => Les fichier de controle
		RequeteJS/
			* => fichier php correspondant aux requetes JS

	/Projet/JavaScript/
		* => Les scripts Javascript
		event* => Les scripts Javascript gérant les évenements

	/Projet/Images
		* => différentes images utilisé dans le site

	/Projet/index.php => page principale gérant les redirections
